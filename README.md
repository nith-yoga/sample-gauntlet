# SAMPLE GAUNTLET - A Sample-based Music Challenge

**Sample Gauntlet** is a one-screen music discovery game built for crate-diggers. Name the correct song based off of the sample source across 20 rounds of increasing difficulty.

---

## Features

*   **Progressive Difficulty:** 5 Tiers of tracks where the sample is meant to be harder to recognize in each tier. Think of it like the difficulty tiers in games like *Rock Band*.
*   **One-Screen UI:** Optimized for a focused, no-scroll experience to mimic a native arcade cabinet.
*   **Responsive Lives System:** 3 lives to clear the gauntlet. 3 wrong guesses and it's game over.
*   **YouTube Integration:** All music & visuals are sourced through YouTube.
*   **Custom Dataset:** Everything from the pool of samples to the difficulty tiers is handpicked, to ensure a genuinely challenging experience.

---

## Tech Stack

- **Framework:** [Next.js 14](https://nextjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Components:** [Fuse.js](https://www.fusejs.io/)
- **Video Engine:** `react-youtube` (YouTube IFrame Player API)
- **Animations:** `tailwindcss-animate`
